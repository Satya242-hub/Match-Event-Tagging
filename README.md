# Match-Event-Tagging

Project Overview
The project simulates the type of tagging work commonly used in sports analytics platforms where raw match footage is converted into structured data for analysis.

Match Information

Dataset Structure

The dataset was created using  Excel with the following columns:

Match_ID – Unique identifier for the match
Half – First or Second half of the match
Minute – Approximate timestamp of the event
Team – Team performing the event
Event_Type – Type of event recorded
Outcome – Result of the event

Event Types Used

PASS – A player passes the ball to a teammate
SHOT – A player attempts to score
SAVE – Goalkeeper blocks or catches a shot
FOUL – A rule violation committed by a player

Outcome Categories

Successful – Action completed successfully
Unsuccessful – Action failed or intercepted
On Target – Shot directed towards the goal
Off Target – Shot missed the goal
Blocked – Shot blocked by defender
Yellow Card – Foul resulting in a yellow card

Result
Created a structured dataset ready for basic sports analysis
